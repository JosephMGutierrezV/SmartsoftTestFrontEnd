## Tecnologias usadas

- Angular
- Bootstrap
- Kendo
- ngrx

## Despliegue

Descargue las fuentes y ejecute en la terminal los siguientes comandos:

- `$ npm install` # Instale las dependencias.
- `$ npm ng serve -o` # Levante el servidor local.
